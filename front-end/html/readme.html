【關於本網頁】

    本網頁由學員 張博裕 製作，僅作為示範用途，旨在建立後續搭建個人網頁之基礎框架。
      展示功能如下
        header：左上角回初始展示頁(本頁)之圖示icon
        導覽列：左/右兩側功能面板收合之漢堡包圖標、外部連結下拉式清單
        左側功能面版(預設展開)：選取並載入對應上課素材至中央展示區
        右側功能面版(預設收合)：選取對應風景圖片，將圖片投放至畫面中央
        footer：點選「聯絡我」字樣自動代入電子信箱觸發mailto，預設為XXXXX@gmail.com
    
      index.html       本專案的入口.html檔                 
  專    |-css          用來存放本專案用到的CSS樣式           
  案    |-fonts        用來存放本專案用到的字型           
  資    |-html         本readme.html的存放位置            
  料    |-icon         用來存放本專案用到的icon           
  夾    |-images       用來存放本專案用到的所有圖片檔      
  結    |-js           用來存放本專案用到的Javascript檔   
  構    |-pages        用來存放上課期間蒐集到的素材檔(HTML、JS、CSS)

*本作品之首頁icon、header背景圖由Gemini生成，其餘素材均為學員個人所有，未經許可請勿擅自轉載。
