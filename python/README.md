## 關於task-v1.1.0.py
此檔為自製python筆記軟體之任務管理小工具，僅作為展演用途，故無法單獨使用。

## 展示功能
1. 任務欄位展示：主要可分為任務、優先級、截止日、狀態、群組五個欄位，並可以根據對應欄位進行任務篩選。
2. 任務排序功能：可選擇使用添加時間排序、按優先級排序、按截止時間排序
3. 管理任務檔功能：可選擇匯出任務、匯入任務、合併任務功能
4. 右鍵選單-變更任務功能：可更改任務名稱、優先級、截止日、群組、狀態
5. 右鍵選單-任務備註功能：可新增任務內容細節以及備註事項
6. 右鍵選單-任務記錄功能：可針對同一項任務進行多次的任務記錄，以達成進度追蹤的效果。
7. 右鍵選單-管理群組功能：可進行新增、刪除、重新命名任務群組
8. 右鍵選單-移除任務功能：可進行任務移除
